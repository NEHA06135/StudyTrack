# StudyTrack – Professional Study Planner

A Java-based desktop application for managing academic tasks with real-time progress tracking and analytics.

## Features
- Task management with priority & deadlines
- Real-time progress tracking (slider)
- Automatic status updates
- CSV-based data persistence
- Analytics dashboard

## Tech Stack
- Java (JDK 11+)
- Swing & AWT
- MVC Architecture

## How to Run
javac StudyPlannerApp.java
java StudyPlannerApp
